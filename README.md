# Advent of code 2017 (aoc 2017)

My C# solutions to http://adventofcode.com/2017 using .NET Core 2.0.

## Dependencies

- This library is based on .NET Core 2.0. It should work on Windows, Linux and OS X.

## Running

To run the project:

1. Install .NET Core.
2. Download the code.
3. Run `dotnet run <day>`.
